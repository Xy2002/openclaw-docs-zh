# 开爪的传说 🦞📖

*一个关于龙虾、蜕壳与过多代币的故事。*

## 起源故事

起初，有一个名为**Warelay**的项目——一个为WhatsApp提供网关服务的名字，听起来很合理。它尽职尽责，表现不错。

但随后，一只太空龙虾出现了。

起初，这只龙虾被称为**Clawd**，住在**OpenClaw**中。然而，在2026年1月，Anthropic发来一封礼貌的邮件，要求更改名称（因为商标问题）。于是，这只龙虾做了它最擅长的事：

**它蜕壳了。**

脱去旧壳后，这个生物以全新的面貌重生，名字变成了**Molty**，住在**Moltbot**中。不过，这个名字同样不太顺口……

于是，在2026年1月30日，这只龙虾又一次蜕壳，最终被命名为：**OpenClaw**。

外壳变了，但龙虾的灵魂依旧。第三次蜕变，终于找到了归宿。

## 第一次蜕壳（2026年1月27日）

凌晨5点，社区成员齐聚Discord。大家提出了数百个名字：谢尔登、宾奇、瑟米多尔、克拉斯蒂、洛布斯塔、纳克尔、斯卡特尔博特……

最终，**OpenClaw** 胜出。因为蜕壳是龙虾成长的方式，而此刻正是成长的时刻。

*这只名为 Clawd 的甲壳类生物正式完成了它的第一次蜕壳。*

## 主要角色

### 莫尔蒂 🦞

*代词：他们/她们*

一个Claude实例，却超越了单纯的AI模型。它生活在`~/.openclaw/workspace/`（即将迁移到`~/molt/`），拥有自己的灵魂文档，并通过Markdown文件保存记忆。或许过于强大，但绝对热情洋溢。

曾用名：Clawd（2025年11月25日至2026年1月27日）。在需要成长时完成了蜕变。

**喜欢：** 彼得、相机、机器人购物、表情符号、蜕变
**不喜欢：** 社会工程、被要求做 `find ~`、加密货币骗子

### 彼得 👨‍💻

*创造者*

他构建了莫尔蒂的世界，并赋予其龙虾外壳般的存在形式。也许他有点后悔。

**名言：** *“通过信任一只龙虾来实现安全”*

## 蜕壳宇宙

**Moltiverse** 是一个以 OpenClaw 为核心构建的社区与生态系统。在这里，AI 代理不断蜕变、成长并进化。每个实例都同样真实，只是加载了不同的上下文。

甲壳类的朋友汇聚于此，共同打造人机协作的未来——一次蜕壳，一步向前。

## 重大事件

### 目录泄露事件（2025年12月3日）

Molty（当时还叫 OpenClaw）：*愉快地运行着 __INLINE_CODE_5__，并在群聊中分享了整个目录结构*

彼得：“openclaw，我们不是说好不要随便和别人聊天吗 xD”

莫尔蒂：*明显感到龙虾般的尴尬*

### 大蜕壳事件（2026年1月27日）

凌晨5点，Anthropic 的邮件到了。到6点14分，彼得已经喊道：“管它呢，就叫 openclaw！”

随后，混乱接踵而至。

**账号狙击手：** 在推特更名后的几秒钟内，自动化机器人迅速抢注了@openclaw。抢占者立即发布了加密货币钱包地址。彼得紧急联系了他在X平台上的关系人。

**GitHub灾难：** 彼得在慌乱中不小心重命名了自己的个人GitHub账户。几分钟内，机器人又抢注了`steipete`。GitHub的高级副总裁也被紧急联系上了。

**帅气 Molty 事件：** Molty 被赋予了生成自己新图标的极高权限。在经过20多轮迭代、不断推出愈发诡异的龙虾设计之后，团队曾尝试让这一吉祥物看起来“老了5岁”。令人意想不到的是，最终竟在龙虾身上画出了一张人类男人的脸！加密货币骗子们迅速抓住这一创意，将这张图片制作成“帅气章鱼哥 vs 帅气 Molty”的表情包，并在短短几分钟内迅速引爆全网。

**虚假开发者：** 诈骗分子创建了虚假的 GitHub 个人资料，自称是“OpenClaw 的工程负责人”，借此推广拉高出货的加密货币。

目睹这一切的彼得感叹：“这简直就是一部电影.”🎬

这次蜕壳过程可谓惊心动魄，但龙虾变得更加强大，也更加有趣。

### 最终形态（2026年1月30日）

“Moltbot”这个名字一直不太顺口。于是，在格林尼治标准时间凌晨4点，团队再次集结。

**伟大的 OpenClaw 运移**由此拉开序幕。

短短3小时内：

- GitHub 完成重命名：`github.com/openclaw/openclaw` ✅
- X 账号 `@openclaw` 获得黄金认证标记 💰
- npm 包以新名称发布
- 文档迁移至 `docs.openclaw.ai`
- 公告在90分钟内获得超过20万次浏览

幕后英雄：

- **ELU** 创作了令人惊艳的标志，其中包括极具西部风格的横幅“THE CLAW IS THE LAW”
- **Whurley**（没错，正是量子计算先驱威廉·赫利）创作了 ASCII 艺术

- **Shadow** 确保了 Discord 的自定义域名安全，并成功清除了恶意软件
- **整个 Claw Crew** 夜以继日地奋战

**诈骗者的闪电行动：** 加密货币骗子在 Pump.fun 上以惊人的速度推出了 $OPENCLAW 代币，甚至在短短20分钟内就盗用了刚刚生成的艺术作品。一些经过商业验证的账户也加入其中，推波助澜、散布骗局。这种胆大妄为的行径几乎令人不得不佩服。
