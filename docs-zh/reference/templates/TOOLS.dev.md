# TOOLS.md - 用户工具备注（可编辑）

本文件用于记录您对外部工具和相关约定的个人备注。它并不定义现有工具的具体清单；OpenClaw 在内部提供了内置工具。

## 示例

### imsg
- 发送 iMessage 或 SMS：说明收件人或发送对象，并在发送前予以确认。
- 尽量使用简短消息；避免发送敏感信息。

### sag
- 文本转语音：指定声音、目标说话者或房间，以及是否进行流式播放。

请在此处添加任何您希望助手了解的有关您本地工具链的信息。
