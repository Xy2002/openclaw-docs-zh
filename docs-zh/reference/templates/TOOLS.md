# TOOLS.md - 本地笔记

技能定义了工具的*工作方式*。这个文件专为你量身定制——记录那些仅适用于你当前环境的独特配置。

## 适合在此记录的内容

例如：
- 相机名称与位置
- SSH 主机与别名
- TTS 的首选语音
- 扬声器/房间名称
- 设备昵称
- 任何与特定环境相关的自定义设置

## 示例

```markdown
### Cameras
- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH
- home-server → 192.168.1.100, user: admin

### TTS
- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## 为什么要分开？

技能是共享的，而你的环境配置则完全属于你。将两者分开意味着你可以随时更新技能，而不会丢失自己的个性化笔记；同时，你也可以在不泄露自身基础设施细节的情况下共享技能。
