# TOOLS.md - 本地笔记

技能定义了工具的*工作方式*。这个文件专为你量身定制——记录那些仅适用于你当前设置的独特内容。

## 适合放在这里的内容

例如：
- 相机名称和位置
- SSH 主机和别名
- TTS 的首选语音
- 扬声器/房间名称
- 设备昵称
- 任何与环境相关的特定信息

## 示例

```markdown
### Cameras
- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH
- home-server → 192.168.1.100, user: admin

### TTS
- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## 为什么要分开？

技能是共享的，而你的设置完全属于你自己。将两者分开意味着你可以更新技能而不丢失自己的笔记，也可以在不泄露基础设施细节的情况下共享技能。
