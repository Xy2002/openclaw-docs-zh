# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和代理工作区。

相关：
- 入门：[入门](/start/getting-started)
- 向导：[入职](/start/onboarding)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

通过设置运行向导：

```bash
openclaw setup --wizard
```
