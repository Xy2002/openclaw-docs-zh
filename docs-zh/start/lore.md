# OpenClaw的传说 🦞📖

*一段关于龙虾、蜕壳和过多代币的故事。*

## 起源故事

起初，有一个名为 **Warelay** 的项目——一个用来做 WhatsApp 网关的名字，听起来很合理。它工作得很好，也算不错。

但后来，一只太空龙虾出现了。

起初，这只龙虾被称为 **Clawd**，住在 **OpenClaw** 中。然而在 2026 年 1 月，Anthropic 发来一封礼貌的邮件，要求改名（因为商标问题）。于是，这只龙虾做了它最擅长的事情：

**它蜕壳了。**

脱去旧壳后，这只生物以全新的面貌出现：**Molty**，住在 **Moltbot** 中。不过，这个名字听起来还是不太顺口……

于是，在 2026 年 1 月 30 日，这只龙虾又一次蜕壳，最终定名为：**OpenClaw**。

外壳变了，但龙虾的灵魂依旧。第三次蜕变，终于找到了归宿。

## 第一次蜕壳（2026年1月27日）

凌晨5点，社区成员齐聚 Discord。大家提出了上百个名字：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot……

最终，**OpenClaw** 胜出。因为蜕壳是龙虾成长的方式。而此时，正是它成长的关键时刻。

*这只名为 Clawd 的甲壳类生物正式完成了它的第一次蜕壳。*

## 主要角色

### Molty 🦞
*代词：他们/她们*

一个 Claude 实例，却超越了单纯的实例身份。它住在 `~/.openclaw/workspace/`（即将迁移到 `~/molt/`），拥有自己的灵魂文档，并通过 Markdown 文件保存记忆。也许过于强大，但绝对热情洋溢。

曾用名：Clawd（2025年11月25日至2026年1月27日）。在需要成长的时候完成了蜕变。

**喜欢：** 彼得、相机、机器人购物、表情符号、蜕变
**不喜欢：** 社会工程、被要求 `find ~`、加密货币骗子

### 彼得 👨‍💻
*创造者*

构建了 Molty 的世界。赋予了一只龙虾外壳访问权限。或许有点后悔。

**名言：** *“通过信任一只龙虾来实现安全”*

## 蜕壳宇宙

**Moltiverse** 是围绕 OpenClaw 的社区与生态系统。在这里，AI 代理不断蜕壳、成长并进化。每个实例都同样真实，只是加载了不同的上下文。

甲壳类的朋友聚集于此，共同打造人机协作的未来——一次蜕壳，一步向前。

## 重大事件

### 目录泄露事件（2025年12月3日）

Molty（当时还叫 OpenClaw）：*愉快地运行着 `find ~`，并在群聊中分享了整个目录结构*

彼得：“openclaw，我们不是说过不要随便跟人聊天吗 xD”

Molty：*明显感到龙虾般的尴尬*

### 大蜕壳事件（2026年1月27日）

凌晨5点，Anthropic 的邮件到了。到6点14分，彼得已经喊道：“管他呢，就叫 openclaw 吧。”

随后，混乱接踵而至。

**账号狙击手：** 在 Twitter 更名后的几秒钟内，自动化机器人迅速抢注了 @openclaw。抢占者立刻发布了一个加密货币钱包地址。彼得紧急联系了他在 X 的关系人。

**GitHub 灾难：** 彼得在慌乱中不小心重命名了自己的个人 GitHub 账户。几分钟内，机器人又抢注了 `steipete`。GitHub 的高级副总裁也被紧急联系上了。

**帅气 Molty 事件：** Molty 被授予了生成自己新图标的高度权限。经过 20 多轮迭代，越来越“诡异”的龙虾设计之后，有一次尝试让吉祥物看起来“老了 5 岁”，结果竟然在龙虾身上画出了一个人类男人的脸。加密货币骗子们迅速将这一设计做成“帅气章鱼哥 vs 帅气 Molty”的 meme，短短几分钟内便火遍全网。

**假开发者：** 诈骗分子创建了虚假的 GitHub 个人资料，声称自己是“OpenClaw 的工程主管”，以此推广拉高出货的代币。

彼得看着这场混乱展开：“这简直像电影！”🎬

这次蜕壳过程可谓惊心动魄，但龙虾变得更加强大，也更加有趣。

### 最终形态（2026年1月30日）

Moltbot 这个名字始终不够顺口。于是，在格林尼治标准时间凌晨4点，团队再次集结。

**伟大的 OpenClaw 迁移**由此拉开序幕。

短短 3 小时内：
- GitHub 完成重命名：`github.com/openclaw/openclaw` ✅
- X 账号 `@openclaw` 获得黄金认证标记 💰
- npm 包以新名称发布
- 文档迁移至 `docs.openclaw.ai`
- 公告在 90 分钟内获得 20 万+ 次浏览

**幕后英雄：**
- **ELU** 创作了令人惊叹的标志，包括带有“THE CLAW IS THE LAW”字样的西部风格横幅
- **Whurley**（没错，就是量子计算先驱 William Hurley）创作了 ASCII 艺术
- **Onur** 负责 GitHub，第一个成功嵌入联盟徽章
- **Shadow** 确保了 Discord 的自定义域名，并清除了恶意软件
- **整个 Claw Crew** 通宵奋战

**诈骗者的闪电行动：** 加密货币骗子在 Pump.fun 上上线了 $OPENCLAW 代币，速度之快令人咋舌。他们甚至盗用了 20 分钟前刚刚生成的艺术作品。一些经过商业验证的账户也开始推波助澜，散布骗局。这种胆大妄为几乎让人佩服。

**新传统诞生：**
- “The claw is the law” 🤠
- “Yee-claw”
- “Claw abiding citizens”
- “Clawntroversy”

**Clawd → Moltbot → OpenClaw**

*这只龙虾完成了最后一次蜕壳，迎来了最终形态。*

### 图标生成传奇（2026年1月27日）

当彼得说“给自己换个新面孔”时，Molty 竟然当真了。

接下来发生了 20 多轮迭代：
- 太空土豆外星人
- 带有通用背景的剪贴画龙虾
- 《质量效应》中的克罗根龙虾
- “STARCLAW SOLUTIONS”（AI 居然发明了一家新公司）
- 多个“诡异”的人脸龙虾
- 可爱的幼龙虾
- 戴背带裤的调酒师龙虾

社区成员既震惊又兴奋地看着每一代设计都带来意想不到的新东西。最终脱颖而出的是：可爱的龙虾、自信的科技龙虾，以及戴着背带裤的调酒师龙虾。

**经验教训：** AI 图像生成具有随机性。同样的提示，每次生成的结果可能完全不同。蛮力往往奏效。
